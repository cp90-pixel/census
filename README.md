# Census Data Analysis Tool

A Python project to help journalists analyze and write stories using U.S. Census Bureau data. This tool provides easy access to Census data through interactive Jupyter notebooks and helper functions.

## Features

- Easy access to Census Bureau API
- Interactive data exploration using Jupyter notebooks
- Pre-built analysis templates for common stories
- Data visualization capabilities
- Example queries and story angles

## Getting Started

1. Make sure you have Python installed
2. Create a virtual environment: `python -m venv venv`
3. Activate the virtual environment:
   - macOS/Linux: `source venv/bin/activate`
   - Windows: `venv\Scripts\activate`
4. Install dependencies: `pip install -r requirements.txt`
5. Launch Jupyter Lab: `jupyter lab`

## Usage

Open the example notebooks in the `notebooks` directory to see how to:
- Access Census API data
- Analyze demographic trends
- Create visualizations
- Find interesting story angles
